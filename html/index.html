The office API RESTful

Esta documentación te va a mostrar las diferentes rutas para obtener personajes de The office.

GET

URL: http://localhost:3000/characters/

Esta URL contiene todos los GET que podemos realizar para obtener personajes de distintas maneras.

  • Characters: Obtenemos todos los personajes
  • Frases: Obtenemos las frases y quien dijo esa frase
  • ID: Obtenemos un personaje por id
  • Nombre: Obtenemos un personaje por su nombre
  • Frase: Podemos buscar mandando por parametro una frase, lo que nos devuelve quien fue el personaje que dijo esa frase
{ "characters": "http://localhost:3000/characters/", "frases": "http://localhost:3000/characters/quotes", "id": "http://localhost:3000/characters/:id", "nombre": "http://localhost:3000/characters/:name", "frase": http://localhost:3000/characters/quote/:quotes }

Filtros

Además podemos filtrar por episodio de aparición y por género del personaje

{ "episodio": "http://localhost:3000/characters/episodes/:first", "genero": "http://localhost:3000/characters/gender/:gender" }

Paginado y ordenamiento de resultados

Tenemos la posibilidad de paginar y ordenar según episodio de aparición

En el caso del paginado, enviamos por query la página que vamos a obtener con la cantidad de resultados a devolver

http://localhost:3000/characters/limit-characters?page=1&limit=3

{ "paginado": "http://localhost:3000/characters/limit-characters", "orden por episodio": "http://localhost:3000/characters/sort-by-episode" }

POST

Para crear personajes, debemos mandar por body name, occupation, first, last, quotes

Quotes es un array que admite agregar varias frases.

URL: http://localhost:3000/characters/

PUT

Tenemos la opción de modificar las frases de los personajes, para eso usamos la URL http://localhost:3000/characters/:id

DELETE

Para eliminar un personaje usamos la URL http://localhost:3000/characters/:id
